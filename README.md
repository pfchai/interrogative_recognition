# interrogative_recognition

识别句子是否为疑问句

## 疑问句分类

- 疑问词问句（特指问句）（question_words），例：你什么时候回来
- 是非问句（whether），例：你今天会准时下课吗
- 选择问句（choice），例：他是坐火车来的，还是坐汽车来的
- 正反问句（pos_and_neg），例：你怕不怕风吹
- 附加问句（attach），例：这是小明的，是不是
- 反问句（contrary），例：难道你不喜欢他吗

## 支持的算法

- [x] 正则
- [ ] 词袋特征 + XGBoost
- [ ] TF-IDF + XGBoost
